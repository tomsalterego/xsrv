# xsrv.common.users

This role will configure Linux user accounts on Debian-based systems.


## Requirements/dependencies/example playbook

See [meta/main.yml](meta/main.yml)

```yaml
# playbook.yml
- hosts: my.CHANGEME.org
  roles:
    - nodiscc.xsrv.common.users
    # - nodiscc.xsrv.common # (alternative) basic debian server configuration including users

```

See [defaults/main.yml](defaults/main.yml) for all configuration variables
